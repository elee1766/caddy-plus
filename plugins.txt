# caddy-plus plugin manifest
#
# One plugin per line:  <go-module-path>@<version>
#
# Rules enforced by the pipeline:
#   - every entry MUST be pinned to an exact version (vX.Y.Z or pseudo-version)
#   - "@latest", branches and commit SHAs are rejected
#   - blank lines and lines starting with # are ignored
#
# The build fails if any plugin listed here is not found inside the
# resulting binary at the exact pinned version.
#
# Examples (uncomment and adjust as desired):
#
# github.com/caddy-dns/cloudflare@v0.2.2
# github.com/caddyserver/cache-handler@v0.16.0
